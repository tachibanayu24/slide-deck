slide-deck

LLMフレンドリーなHTMLスライドフレームワーク

2026年4月 — Phase 1 プロトタイプ

特徴

  • HTMLが一級市民 — Markdown変換ではなく、HTML/CSS/SVGを直接記述
  • Web Components — <s-deck> + <s-slide> で構造を定義
  • PDF出力 — ブラウザの印刷機能で完結。16:9、余白なし
  • Tailwind CSS — LLMが既に知っているユーティリティで自由にスタイリング

プラグインをインストールするだけで、高品質なスライドが手に入る

2層アーキテクチャ

フレームワークとプラグインを分離。ロックインなし。

npm

フレームワーク

CDN配信。ブラウザが消費

plugin

プラグイン

Skills / Commands。Claudeが消費

レイアウトプリセット

layout 属性で切り替え:

title タイトル
text テキスト主体
text-image 左文 + 右図
two-column 2カラム
grid-2x2 2×2 グリッド
section セクション区切り

フレームワーク API

// これだけ覚えればOK
<s-deck>
<s-slide
layout="title"
theme="dark"
bg="...">
// Tailwind で自由に
</s-slide>
</s-deck>
