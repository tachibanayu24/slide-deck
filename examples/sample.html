Layouts

10 presets for every slide type

Layout Presets

title Cover / closing
text Body text / bullets
text-image Left text + right visual
image-text Left visual + right text
two-column Side by side
grid-2x2 4-quadrant matrix
chart Text + chart area
section Chapter divider
toc Table of contents
full-image Full visual

Minimal API

// That's all you need
<s-deck>
<s-slide
layout="title"
theme="dark"
bg="...">
// Tailwind as usual
</s-slide>
</s-deck>

+ Lucide icons via <i data-lucide="...">

grid-2x2

4 equal quadrants. Great for SWOT, comparisons, feature matrices.

theme="dark"

Auto light text. Just add the attribute.

data-area

"text" and "visual" roles for grid layouts.

section attr

Footer shows section name. Hidden on title slides.
